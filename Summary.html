  • A set A is said to be subset of a set B, if every element of A is also an element of B. Intervals are subsets of R.
  • A power set of a set A is collection of all subsets of A. It is denoted by P(A).
  • The union of two sets A and B is the set of all those elements which are either in A or in B.
  • The intersection of two sets A and B is the set of all elements which are common. The difference of two sets A and B in this order is the set of elements which belong to A but not to B.
  • The complement of a subset A of universal set U is the set of all elements of U which are not the elements of A.
  • For any two sets A and B,\((A \cup B)'=A' \cup B'\)
  • If A and B are finite sets such that \(A \cap B = \phi, \text{ then } n(A \cup B)= n(A)+N(B)\).
  • \(\text{ If } A \cap B \neq \phi, \text{ then } n(A \cup B)=n(A) + n(B) - n(A \cap B)\)

    • Some important results on subsets

    1. Every set is a subset of itself.

    2. The empty set is a subset of every set.

    3.The total number of subsets of a finite set containing \(n\) elements is \(2^n\) and the total number of proper subsets is \(2^n - 1\).

    4.\( \phi\) has no proper subset.

    Algebra of Operations

    1. Commutative Laws

    \((i) A \cup B = B \cup A\qquad (ii)A \cap B = B \cap A \)

    2. Associative Laws

    \( (i) (A \cup B)\cup C = A \cup (B \cup C) \qquad (ii) (A \cap B)\cap C = A \cap (B \cap C)\)

    3. Identity Laws

    \( (i) A \cup \phi = A \qquad (ii) A \cap \phi =\phi\)

    4. IdempotentLaws

    \( (i) A \cup A =A \qquad (ii)A \cap A = A\)

    5. Distributive Laws

    \((i) A \cap (B \cup C) =(A \cap B) \cup (A \cap C)\)

    \( (ii) A \cup (B \cap C) = (A \cup B) \cap (A \cup C)\)

    Laws on Complement of a Set

    1. Involution Law

    \( (A')'=A.\)

    2. Complement Laws

    \( (i) A \cup A'= U \qquad (ii)A \cap A'= \phi \qquad (iii)\phi' = U \qquad (iv) U'=\phi \)

    3.De Morgan's Laws

    \( (i) (A \cup B)'= A' \cap B' \qquad (ii)(A \cap B)'=A'\cup B'\)
