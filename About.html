Important of set

Set theory is important because it is a theory of integers, models of axiom systems, infinite ordinals, and real numbers, all in one unified structure. This allows it to serve as a foundation for all mathematics, anything about in mathematics can be formalized in set theory naturally and easily, and studing set theory allows us to prove theorems about in mathematics itself.

Definition:

A set is well defined collection of objects. The objects of the set are called elements or members of the set. A set is usually denoted by capital letter while its elements are denoted by small letters.

Representation of a set:

(i) Roster or tabular form

(ii) Set-builder form

Type of Sets

\( (i) \) Empty Set

Empty set is also called a null set or void set. It is denoted by {} or \( \phi \).

\( (ii) \) Finite Set

A set which contains finite number of elements is called a finite set.

\( (iii) \) Infinite Set

A set which is not finite or which contains infinite numbers of elements is called an infinite set.

\( (iv) \) Singleton Set

A set which contains only one element is called Singleton Set

\( (v) \) Equal Set

Two sets are said to be Equal Sets if they have same elements.

\( (vi) \)Equivalent Set

The two sets A and B are said to be Equivalent Set if \( n(A) =n(B)\).
